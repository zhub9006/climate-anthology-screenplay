# 🌍 Climate Anthology Screenplay

A collaborative climate-change anthology screenplay written by our screenwriting class. The project is structured around three story segments, each exploring a different facet of the climate crisis.

## 🎬 Story Segments

| Segment | Theme | Directory |
|---------|-------|-----------|
| **Rising Seas** | Coastal flooding, displacement, sea-level rise | `screenplays/rising-seas.md` |
| **Wildfires** | Megafires, smoke, loss of wilderness | `screenplays/wildfires.md` |
| **Drought** | Water scarcity, heatwaves, agricultural collapse | `screenplays/drought.md` |

## 📂 Repository Structure

```
climate-anthology-screenplay/
├── README.md
├── research/
│   └── environmental-data.md      # Live weather & AQI data for Dhaka
├── screenplays/

│   ├── wildfires.md               # Wildfires screenplay
│   └── drought.md                 # Drought screenplay
```

## 📝 How to Contribute

1. **Pick a segment** — check the GitHub Issues for your assigned story.
2. **Read the research** — each issue contains curated scientific references and news summaries.
3. **Write in your branch** — create a feature branch from `main` (e.g., `rising-seas/draft`).
4. **Open a Pull Request** — tag your classmates for review.
5. **Iterate** — incorporate feedback, ground your scenes in the research data.

## 🔬 Research Sources

- IPCC AR6 Working Group Reports (2021–2023)
- NASA Climate Change Vital Signs
- Nature: Global increase in wildfire potential from compound fire weather
- ScienceDirect: Climate change impacts on extreme weather events
- UN Climate Reports on impacts affecting 3.6 billion people

See `research/environmental-data.md` for live Dhaka environmental data used as reference.

## 🤝 Guidelines

- Use **standard screenplay format** (Courier 12pt, scene headings, action lines, dialogue).
- Anchor fictional scenes in **real scientific findings** — see issue research summaries.
- Each segment should be **15–25 pages** (approx. 15–25 minutes of screen time).
- Collaborate across segments for tonal consistency and shared characters.
